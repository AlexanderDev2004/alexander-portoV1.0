# üèòÔ∏è Jawara (RuangWarga)

Jawara is a neighborhood (RT) management mobile app built with Flutter and a FastAPI backend. It helps citizens and admins handle daily community tasks in one place.

---

## ‚ú® Key Features

**For Citizens**
- Dashboard with announcements and quick actions.
- Marketplace for buying and selling items.
- Monthly fee payments and transaction history.
- Digital document requests (domicile/business letters).
- Issue reporting (security, infrastructure, etc.).
- Community activities and events.
- Profile and family data management.

**For Admins**
- Admin dashboard with statistics and overview.
- Approve/reject new registrations.
- Manage banners and announcements.
- Full CRUD for activities (with photo uploads).
- Review document requests and citizen reports.
- Finance monitoring and fee management.
- Marketplace transaction oversight.
